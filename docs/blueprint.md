# **App Name**: AetherLease

## Core Features:

- Multilingual Support: Implement i18n to allow users to switch between different languages (polish and english).
- About Us Section: A dedicated section providing information about the platform and its mission.
- Main Landing Section: The primary section showcasing the platform's key features and value proposition with eye-catching motion design
- Resources Hub: Display links to essential resources such as GitHub repositories and SDK documentation in an organized, card-based layout.
- Terms and Policies: Provide clear and easily accessible links to the platform's Terms of Service, Privacy Policy, and other legal documents.
- Server Marketplace Preview: Display a demo server marketplace, accessible only behind a login wall. This marketplace will show example server configurations and pricing without enabling actual transactions.

## Style Guidelines:

- Primary color: A deep navy blue (#2E3148) evoking trust and stability.
- Background color: Light gray (#E8E8E8), creating a clean, modern feel.
- Accent color: A vibrant teal (#00A3AD) for interactive elements and calls to action.
- Body: 'PT Sans' a modern and friendly sans-serif font. Headlines: 'Playfair' for a modern sans-serif style with thin-thick lines, for a high-end feel. 
- Employ a clean, grid-based layout to ensure a professional and well-organized presentation of information. Optimize for various screen sizes to provide a consistent user experience on desktop and mobile devices.
- Incorporate subtle animations using Anime.js or GSAP to enhance user engagement. Focus on smooth transitions and micro-interactions that add a touch of sophistication without overwhelming the user.
